INTRODUCTION
------------
Are you a developer and you miss an Export button on
Configuration > Configuration management > Export > Single item form
(/admin/config/development/configuration/single/export)? Then this module is for
you. It alters the form and adds an Export button. Clicking on that button you
can download the selected configuration to a folder of your choice.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/config_single_export

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/config_single_export?categories=All


REQUIREMENTS
------------
No special requirements.


INSTALLATION
------------
 * Install as you would normally install a contributed Drupal module. See:
   https://www.drupal.org/documentation/install/modules-themes/modules-8
   for further information.


SET-UP
------
The module has no setting page or modifiable options.


HOW TO USE
----------
After enabling the module an Export button will appear on the bottom of
/admin/config/development/configuration/single/export page. Select the
configuration type and name you want to export then click on the Export button.
The configuration file will be downloaded to the folder of your choice.
